## What makes special

Alseek is a Responsive Flat Admin CRM and written with Vue and SASS.

- Responsive layout (desktops, tablets, mobile devices)
- Built with Bootstrap v4.3.1
- Clean and Flat design
- Vue version
- Dashboard
- Categories
- Items
- Orders
- Promocodes
- Reviews
- Users
- Roles
- Site Info
- Currency
